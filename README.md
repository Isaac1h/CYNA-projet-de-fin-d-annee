# 📂 CYNA - Documentation

Cette branche contient tous les documents de conception et de gestion du projet.

### 📁 Contenu de la branche
* **`/note-de-cadrage`** : Objectifs, planning et périmètre du projet.
* **`/sujet-projet`** : Énoncé officiel et contraintes techniques.
* **`/documentation-stripe`** : Ressources pour l'intégration du paiement.
* **`/maquette-figma`** : Maquettes et prototypes UI/UX réalisés sur Figma.

---

### 📌 Règles
- Docs importantes (utiles tout le projet) ➝ **à push ici**  
- Docs temporaires/obsolètes ➝ **à push sur la branche `sprint-x` du sprint concerné**
