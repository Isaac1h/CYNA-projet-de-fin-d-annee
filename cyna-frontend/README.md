# CYNA - Interface Client (SPA)

Ce dossier contient la partie Front-end du projet CYNA. 
C'est une application React moderne construite avec **Vite** et stylisée avec **Bootstrap**.

## 🛠️ Stack Technique
- **Framework :** React 18
- **Build Tool :** Vite
- **Styles :** Bootstrap & React-Bootstrap
- **Icons :** Lucide React
- **Routing :** React Router Dom

## 🚀 Lancement rapide
1. Installer les dépendances : `npm install`
2. Lancer le projet en local : `npm run dev`
